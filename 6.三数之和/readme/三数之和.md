

# 15. 三数之和 - 详细解题分析

## 一. 题目

给定一个整数数组 `nums`，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足：
- `i != j`、`i != k` 且 `j != k`
- `nums[i] + nums[j] + nums[k] == 0`

请你返回所有和为 **0** 且 **不重复** 的三元组。

**注意**：答案中不可以包含重复的三元组。

### 示例

**示例 1：**
```
输入: nums = [-1,0,1,2,-1,-4]
输出: [[-1,-1,2],[-1,0,1]]
```

**示例 2：**
```
输入: nums = [0,1,1]
输出: []
```

**示例 3：**
```
输入: nums = [0,0,0]
输出: [[0,0,0]]
```

### 提示

- `3 <= nums.length <= 3000`
- `-10^5 <= nums[i] <= 10^5`

---

## 二. 解题思路

### 1. 问题分析

**目标**：找出所有和为 0 的三元组，且不能有重复

**关键约束**：
- 三个数的索引必须互不相同
- 三数之和必须等于 0
- 结果中不能包含重复的三元组

**核心挑战**：
- 如何避免重复的三元组？
- 如何高效地找出所有满足条件的三元组？

**核心思想**：
- **排序 + 双指针**：先对数组排序，然后固定一个数，用双指针在剩余部分找两数之和为该数的相反数
- **去重策略**：通过跳过相同的数来避免重复

---

### 解法一：暴力枚举法（容易理解但低效）

#### 思路

三层循环，枚举所有可能的三元组合，检查是否和为 0。

```python
def threeSum(nums: list[int]) -> list[list[int]]:
    """
    三数之和 - 暴力枚举法
    
    时间复杂度：O(n³)
    空间复杂度：O(1)
    
    说明：
        - 三层循环枚举所有三元组
        - 检查和是否为0
        - 使用集合去重
    """
    n = len(nums)
    result = set()  # 使用集合去重
    
    for i in range(n):
        for j in range(i + 1, n):
            for k in range(j + 1, n):
                if nums[i] + nums[j] + nums[k] == 0:
                    # 将三元组排序后转为元组，存入集合
                    triplet = tuple(sorted([nums[i], nums[j], nums[k]]))
                    result.add(triplet)
    
    return [list(triplet) for triplet in result]
```

#### 图解过程

```
数组：[-1, 0, 1, 2, -1, -4]

枚举过程（部分）：

第1轮：i=0, j=1, k=2
  nums[0] + nums[1] + nums[2] = -1 + 0 + 1 = 0 ✓
  添加：(-1, 0, 1)

第2轮：i=0, j=1, k=3
  nums[0] + nums[1] + nums[3] = -1 + 0 + 2 = 1 ✗

第3轮：i=0, j=1, k=4
  nums[0] + nums[1] + nums[4] = -1 + 0 + (-1) = -2 ✗

...

最终找到：
  (-1, -1, 2)
  (-1, 0, 1)

问题：时间复杂度太高！
```

#### 优缺点

- 优点：代码简单，容易理解
- 缺点：**时间复杂度 O(n³)**，无法通过大规模测试

### 解法二：排序 + 双指针法（推荐！）

#### 思路

1. **对数组排序**：方便去重和双指针操作
2. **固定一个数** `nums[i]`：作为三元组的第一个数
3. **双指针找两数之和**：在 `i+1` 到 `n-1` 的范围内，找两个数之和为 `-nums[i]`
4. **去重处理**：跳过相同的数，避免重复三元组

```python
def threeSum(nums: list[int]) -> list[list[int]]:
    """
    三数之和 - 排序 + 双指针法
    
    时间复杂度：O(n²)
    空间复杂度：O(1)（不计结果存储）
    
    算法步骤：
        1. 对数组排序
        2. 遍历数组，固定一个数 nums[i]
        3. 使用双指针 left 和 right 找两数之和
        4. 根据和的大小移动指针
        5. 跳过重复元素去重
    """
    n = len(nums)
    result = []
    
    # 边界情况
    if n < 3:
        return result
    
    # 对数组排序（关键步骤！）
    nums.sort()
    
    # 遍历数组，固定第一个数
    for i in range(n - 2):
        # 跳过重复的 nums[i]，避免重复三元组
        if i > 0 and nums[i] == nums[i - 1]:
            continue
        
        # 如果最小的三数之和都大于0，无需继续
        if nums[i] + nums[i + 1] + nums[i + 2] > 0:
            break
        
        # 如果最大的三数之和都小于0，跳过这个 nums[i]
        if nums[i] + nums[n - 1] + nums[n - 2] < 0:
            continue
        
        # 双指针查找
        left = i + 1
        right = n - 1
        
        while left < right:
            total = nums[i] + nums[left] + nums[right]
            
            if total == 0:
                # 找到一个满足条件的三元组
                result.append([nums[i], nums[left], nums[right]])
                
                # 跳过重复的 left
                while left < right and nums[left] == nums[left + 1]:
                    left += 1
                
                # 跳过重复的 right
                while left < right and nums[right] == nums[right - 1]:
                    right -= 1
                
                # 移动指针继续查找
                left += 1
                right -= 1
            
            elif total < 0:
                # 总和小于0，左指针右移增大和
                left += 1
            
            else:
                # 总和大于0，右指针左移减小和
                right -= 1
    
    return result
```

#### 为什么排序 + 双指针法高效？

```
核心原理：有序数组 + 双指针

假设已排序数组：[a, b, c, d, e, f]
                    ↑      ↑
                   left   right

特点：
  - left 右移 → 总和增大
  - right 左移 → 总和减小
  
这就形成了一个可控制的搜索空间！

二分查找类似的思想，但更直接！
```

### 解法三：哈希表法

#### 思路

1. 遍历数组，固定两个数
2. 用哈希表查找第三个数为 `-sum`
3. 去重处理

```python
def threeSum(nums: list[int]) -> list[list[int]]:
    """
    三数之和 - 哈希表法
    
    时间复杂度：O(n²)
    空间复杂度：O(n)
    
    说明：
        - 先对数组排序
        - 固定两个数，用哈希表找第三个数
        - 需要额外的去重处理
    """
    n = len(nums)
    result = []
    
    if n < 3:
        return result
    
    # 排序
    nums.sort()
    
    for i in range(n - 2):
        # 跳过重复
        if i > 0 and nums[i] == nums[i - 1]:
            continue
        
        # 使用哈希表找两数之和
        seen = set()
        for j in range(i + 1, n):
            complement = -nums[i] - nums[j]
            
            if complement in seen:
                # 找到一个满足条件的三元组
                result.append([nums[i], complement, nums[j]])
                
                # 跳过重复的 nums[j]
                while j < n - 1 and nums[j] == nums[j + 1]:
                    j += 1
            
            # 将当前数加入哈希表
            seen.add(nums[j])
    
    return result
```

## 三. 方法对比

| 方法 | 时间复杂度 | 空间复杂度 | 去重难度 | 推荐程度 |
|------|-----------|-----------|---------|---------|
| **暴力枚举** | O(n³) | O(1) | 简单 | 不推荐 |
| **排序 + 双指针** | O(n²) | O(1) | 简单 | **强烈推荐** |
| **哈希表法** | O(n²) | O(n) | 较复杂 | 一般 |

### 复杂度分析

```
暴力枚举法：
  - 时间：O(n³) - 三层循环
  - 空间：O(1) - 只用常数额外空间
  - n = 3000 时，约 27 亿次运算！

排序 + 双指针法：
  - 时间：O(n²) - 外层循环 O(n) × 内层双指针 O(n)
  - 空间：O(1) - 只用几个指针变量
  - n = 3000 时，约 900 万次运算

性能提升：30000倍！
```

### 为什么排序 + 双指针是最优解？

```
1. 利用有序性：
   - 排序后，相等的数会相邻
   - 双指针可以快速调整和的大小

2. 去重自然：
   - 跳过相同的数即可避免重复
   - 不需要额外的数据结构

3. 空间最优：
   - 只需 O(1) 额外空间
   - 适合内存受限的场景

4. 时间优秀：
   - O(n²) 是三数之和问题的最优时间复杂度
   - 无法做到 O(n log n)
```

## 五. 小贴士

### 1. 为什么必须先排序？

```
面试官可能的考察点：

排序的重要性：
  1. 去重更容易：相等的数会相邻
  2. 双指针可行：有序数组才能用双指针
  3. 剪枝优化：可以提前终止循环

不排序的问题：
  - 无法判断是否重复
  - 无法使用双指针
  - 去重逻辑复杂
```

### 2. 去重的三种策略

```
策略1：跳过相同的 nums[i]
  for i in range(n):
      if i > 0 and nums[i] == nums[i - 1]:
          continue

策略2：跳过相同的 nums[left]
  while left < right and nums[left] == nums[left + 1]:
      left += 1

策略3：跳过相同的 nums[right]
  while left < right and nums[right] == nums[right - 1]:
      right -= 1
```

### 3. 进阶要求的优化

```
剪枝优化：
  1. nums[i] + nums[i+1] + nums[i+2] > 0
     → 最小的三数之和都大于0，后续不可能有解
  
  2. nums[i] + nums[n-1] + nums[n-2] < 0
     → 即使加上最大的两个数都小于0，跳过这个 nums[i]
```

### 4. 与两数之和的区别

```
两数之和：
  - 找一对数，和为目标值
  - 用哈希表：O(n) 时间，O(n) 空间
  - 无需去重（题目不要求）

三数之和：
  - 找三个数，和为0
  - 排序 + 双指针：O(n²) 时间，O(1) 空间
  - 必须去重（题目要求）
```

## 六. 完整代码实现

### Python（排序 + 双指针法 - 最优解）

```python
from typing import List

class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        """
        三数之和 - 排序 + 双指针法
        
        时间复杂度：O(n²)
        空间复杂度：O(1)
        
        算法步骤：
            1. 对数组进行排序
            2. 遍历数组，固定第一个数 nums[i]
            3. 使用双指针 left 和 right 找两数之和
            4. 根据和的大小移动指针
            5. 跳过重复元素避免重复三元组
            6. 剪枝优化提前终止
        
        Args:
            nums: 输入的整数数组
            
        Returns:
            List[List[int]]：所有和为0的不重复三元组
        """
        n = len(nums)
        result = []
        
        # 边界情况：数组长度小于3
        if n < 3:
            return result
        
        # 对数组排序（关键步骤！）
        nums.sort()
        
        # 遍历数组，固定第一个数
        for i in range(n - 2):
            # 跳过重复的 nums[i]，避免重复三元组
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            
            # 剪枝：如果最小的三数之和都大于0，无需继续
            if nums[i] + nums[i + 1] + nums[i + 2] > 0:
                break
            
            # 剪枝：如果最大的三数之和都小于0，跳过这个 nums[i]
            if nums[i] + nums[n - 1] + nums[n - 2] < 0:
                continue
            
            # 双指针查找
            left = i + 1
            right = n - 1
            
            while left < right:
                total = nums[i] + nums[left] + nums[right]
                
                if total == 0:
                    # 找到一个满足条件的三元组
                    result.append([nums[i], nums[left], nums[right]])
                    
                    # 跳过重复的 left
                    while left < right and nums[left] == nums[left + 1]:
                        left += 1
                    
                    # 跳过重复的 right
                    while left < right and nums[right] == nums[right - 1]:
                        right -= 1
                    
                    # 移动指针继续查找
                    left += 1
                    right -= 1
                
                elif total < 0:
                    # 总和小于0，左指针右移增大和
                    left += 1
                
                else:
                    # 总和大于0，右指针左移减小和
                    right -= 1
        
        return result
```

### Python（暴力枚举法 - 容易理解）

```python
from typing import List

class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        """
        三数之和 - 暴力枚举法
        
        时间复杂度：O(n³)
        空间复杂度：O(1)
        
        说明：简单易懂，但效率低
        """
        n = len(nums)
        result = set()
        
        for i in range(n):
            for j in range(i + 1, n):
                for k in range(j + 1, n):
                    if nums[i] + nums[j] + nums[k] == 0:
                        triplet = tuple(sorted([nums[i], nums[j], nums[k]]))
                        result.add(triplet)
        
        return [list(triplet) for triplet in result]
```

### C++（排序 + 双指针法）

```cpp
#include <vector>
#include <algorithm>

class Solution {
public:
    /**
     * 三数之和 - 排序 + 双指针法
     * 
     * 时间复杂度：O(n²)
     * 空间复杂度：O(1)
     */
    std::vector<std::vector<int>> threeSum(std::vector<int>& nums) {
        std::vector<std::vector<int>> result;
        int n = nums.size();
        
        if (n < 3) {
            return result;
        }
        
        // 排序
        std::sort(nums.begin(), nums.end());
        
        for (int i = 0; i < n - 2; i++) {
            // 跳过重复
            if (i > 0 && nums[i] == nums[i - 1]) {
                continue;
            }
            
            // 剪枝
            if (nums[i] + nums[i + 1] + nums[i + 2] > 0) {
                break;
            }
            
            if (nums[i] + nums[n - 1] + nums[n - 2] < 0) {
                continue;
            }
            
            // 双指针
            int left = i + 1;
            int right = n - 1;
            
            while (left < right) {
                int total = nums[i] + nums[left] + nums[right];
                
                if (total == 0) {
                    result.push_back({nums[i], nums[left], nums[right]});
                    
                    while (left < right && nums[left] == nums[left + 1]) {
                        left++;
                    }
                    
                    while (left < right && nums[right] == nums[right - 1]) {
                        right--;
                    }
                    
                    left++;
                    right--;
                }
                else if (total < 0) {
                    left++;
                }
                else {
                    right--;
                }
            }
        }
        
        return result;
    }
};
```

## 八. 总结

### 核心要点

1. **排序是关键**：排序后才能有效去重和双指针
2. **双指针搜索**：将两数之和问题转化为有序数组的双指针查找
3. **三种去重**：分别处理 nums[i]、nums[left]、nums[right] 的重复
4. **剪枝优化**：提前终止不必要的循环

### 算法模板

```python
def threeSum(nums):
    nums.sort()  # 排序
    result = []
    
    for i in range(len(nums) - 2):
        # 去重 + 剪枝
        if i > 0 and nums[i] == nums[i - 1]:
            continue
        
        # 双指针
        left, right = i + 1, len(nums) - 1
        
        while left < right:
            total = nums[i] + nums[left] + nums[right]
            
            if total == 0:
                result.append([nums[i], nums[left], nums[right]])
                # 去重
                while left < right and nums[left] == nums[left + 1]:
                    left += 1
                while left < right and nums[right] == nums[right - 1]:
                    right -= 1
                left += 1
                right -= 1
            elif total < 0:
                left += 1
            else:
                right -= 1
    
    return result
```
