# 11. 滑动窗口最大值 - 详细解题分析

## 一. 题目

给你一个整数数组 `nums`，有一个大小为 `k` 的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的 `k` 个数字。滑动窗口每次只向右移动一位。

返回滑动窗口中的最大值。

### 示例

**示例 1：**
```
输入: nums = [1,3,-1,-3,5,3,6,7], k = 3
输出: [3,3,5,5,6,7]
解释: 
  滑动窗口的位置                最大值 
  ---------------               ----- 
  [1  3  -1] -3  5  3  6  7       3 
   1 [3  -1  -3] 5  3  6  7       3 
   1  3 [-1  -3  5] 3  6  7       5 
   1  3  -1 [-3  5  3] 6  7       5 
   1  3  -1  -3 [5  3  6] 7       6 
   1  3  -1  -3  5 [3  6  7]      7 
```

**示例 2：**
```
输入: nums = [1], k = 1
输出: [1]
```

### 提示

- `1 <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`
- `1 <= k <= nums.length`

---

## 二. 解题思路

### 1. 问题分析

**目标**：找出每个滑动窗口中的最大值

**关键约束**：
- 滑动窗口大小固定为 `k`
- 窗口每次向右移动一位
- 需要返回所有窗口的最大值

**核心思想**：
- **单调递减队列**：维护一个单调递减的队列
- **队列存储索引**：通过索引判断元素是否在窗口内
- **O(n) 时间复杂度**：每个元素最多入队和出队一次

---

### 解法一：单调递减队列法（推荐）

#### 思路

使用单调递减队列（Deque）存储窗口中的元素，保证队首始终是当前窗口的最大值。

```python
from typing import List
from collections import deque

class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        """
        滑动窗口最大值 - 单调递减队列法

        时间复杂度：O(n)
        空间复杂度：O(k)

        算法步骤：
            1. 使用双端队列存储元素的索引
            2. 队列中的元素按值单调递减排列
            3. 队首始终是当前窗口的最大值
            4. 每次移动窗口时：
               - 添加新元素，移除比它小的元素
               - 移除超出窗口范围的元素（索引 < i-k+1）
        """
        if not nums or k == 0:
            return []
        
        result = []
        deque_index = deque()
        
        for i in range(len(nums)):
            # 1. 添加新元素：移除队列中所有比当前元素小的元素
            while deque_index and nums[deque_index[-1]] < nums[i]:
                deque_index.pop()
            
            # 2. 添加当前元素的索引
            deque_index.append(i)
            
            # 3. 移除超出窗口范围的元素
            if deque_index[0] <= i - k:
                deque_index.popleft()
            
            # 4. 当窗口形成后，记录最大值
            if i >= k - 1:
                result.append(nums[deque_index[0]])
        
        return result
```

#### 为什么单调递减队列法可行？

```
核心思想：

  1. 队列的单调性：
     - 队列中的元素按值单调递减
     - 队首始终是当前窗口的最大值

  2. 入队操作：
     - 新元素入队前，移除所有比它小的元素
     - 因为这些元素永远不会成为最大值
     - 新元素可能成为后续窗口的最大值

  3. 出队操作：
     - 当窗口向右移动时，超出窗口范围的元素需要移除
     - 通过索引判断：索引 < i-k+1 表示已不在窗口内

  4. 维护不变量：
     - deque_index 中只包含当前窗口内的元素索引
     - 元素值按单调递减排列
     - 队首就是当前窗口的最大值

  5. 时间复杂度分析：
     - 每个元素最多入队一次
     - 每个元素最多出队一次
     - 总共 O(n) 次操作
     - 时间复杂度 O(n)
```

### 解法二：笨办法（暴力枚举）

#### 思路

对每个滑动窗口，遍历窗口内的所有元素，找到最大值。

```python
from typing import List

class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        """
        滑动窗口最大值 - 暴力枚举法

        时间复杂度：O(n*k)
        空间复杂度：O(1)

        说明：
            - 对每个滑动窗口
            - 遍历窗口内的所有元素
            - 找到最大值
        """
        if not nums or k == 0:
            return []
        
        result = []
        n = len(nums)
        
        for i in range(n - k + 1):
            max_val = nums[i]
            for j in range(i + 1, i + k):
                max_val = max(max_val, nums[j])
            result.append(max_val)
        
        return result
```

#### 复杂度分析

```
暴力枚举法：
  时间复杂度：O(n*k)
    - 窗口个数：n-k+1
    - 每个窗口需要遍历 k 个元素
    - 总共 (n-k+1)*k ≈ O(n*k)

  空间复杂度：O(1)
    - 只使用了常数个变量

单调递减队列法：
  时间复杂度：O(n)
    - 每个元素最多入队和出队一次

  空间复杂度：O(k)
    - 队列最多存储 k 个元素
```

### C++ 实现

```cpp
#include <deque>
#include <vector>
#include <iostream>

using namespace std;

class Solution {
public:
    vector<int> maxSlidingWindow(vector<int>& nums, int k) {
        vector<int> result;
        deque<int> deque_index;
        
        for (int i = 0; i < nums.size(); i++) {
            // 移除比当前元素小的元素
            while (!deque_index.empty() && nums[deque_index.back()] < nums[i]) {
                deque_index.pop_back();
            }
            
            // 添加当前元素的索引
            deque_index.push_back(i);
            
            // 移除超出窗口范围的元素
            if (deque_index.front() <= i - k) {
                deque_index.pop_front();
            }
            
            // 记录窗口最大值
            if (i >= k - 1) {
                result.push_back(nums[deque_index.front()]);
            }
        }
        
        return result;
    }
};
```

## 五. 单调队列的深入理解

### 1. 什么是单调队列？

**单调队列**是一种特殊的数据结构，它维护一个有序的队列，队列中的元素按某种顺序（递增或递减）排列。

**在这个问题中**：我们使用单调递减队列，队首始终是最大值。

### 2. 为什么叫"滑动窗口最大值"？

```
窗口滑动过程：

  窗口1：[1, 3, -1] → 最大值 = 3
  窗口2：[3, -1, -3] → 最大值 = 3
  窗口3：[-1, -3, 5] → 最大值 = 5
  窗口4：[-3, 5, 3] → 最大值 = 5
  窗口5：[5, 3, 6] → 最大值 = 6
  窗口6：[3, 6, 7] → 最大值 = 7

  结果：[3, 3, 5, 5, 6, 7]
```

### 3. 单调队列的三个操作

```python
# 1. 入队前的清理（保持单调性）
while deque and nums[deque[-1]] < nums[i]:
    deque.pop()

# 2. 移除过期元素（超出窗口范围）
if deque[0] <= i - k:
    deque.popleft()

# 3. 添加新元素
deque.append(i)
```

### 4. 直观理解

**类比**：想象你在排队买票，窗口大小为 k。

- **单调递减队列**：身高从高到低排队
- **队首**：最高的人
- **新人加入**：比队尾矮就插后面，比队尾高就插到前面（把挡在前面的人都挤走）
- **窗口移动**：队首离开窗口就出队

## 六. 总结

| 解法 | 时间复杂度 | 空间复杂度 | 说明 |
|------|-----------|-----------|------|
| 单调递减队列 | O(n) | O(k) | 推荐解法 |
| 暴力枚举 | O(n*k) | O(1) | 简单但效率低 |

**关键点：**
1. 理解单调队列的特性：队列中的元素按值单调递减
2. 掌握三个核心操作：入队清理、移除过期元素、添加新元素
3. 队首始终是当前窗口的最大值
4. 每个元素最多入队和出队一次，保证 O(n) 时间复杂度

**应用场景：**
- 滑动窗口问题
- 动态规划中的优化
- 流数据中的最大值/最小值查询
