# 13. 最大子数组和 - 详细解题分析

## 一. 题目

给定一个整数数组 `nums`，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

子数组 是数组中的一个连续部分。

### 示例

**示例 1：**
```
输入: nums = [-2,1,-3,4,-1,2,1,-5,4]
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6。
```

**示例 2：**
```
输入: nums = [1]
输出: 1
解释: 只有一个元素，最大和就是 1。
```

**示例 3：**
```
输入: nums = [5,4,-1,7,8]
输出: 23
解释: 连续子数组 [5,4,-1,7,8] 的和最大，为 23。
```

### 提示

- `1 <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`

**进阶**：如果已经实现复杂度为 O(n) 的解法，尝试使用更为精妙的分治法求解。

---

## 二. 解题思路

### 1. 问题分析

**目标**：在数组中找到连续子数组的最大和

**关键约束**：
- 子数组必须是连续的
- 子数组至少包含一个元素
- 数组可能有正数、负数或零
- 如果所有元素都是负数，应该选择最大的那个负数

**核心思想**：
- **动态规划/Kadane算法**：使用O(n)时间复杂度的经典算法
- **分治法**：使用递归实现，符合进阶要求
- **核心洞察**：前面的子数组和如果是负数，对后续和没有正向贡献，应该丢弃

---

## 三. 解法一：Kadane算法（动态规划/贪心）【推荐】

### 1. 算法思想

Kadane算法的核心思想是：**比较"当前元素"和"前面的子数组和 + 当前元素"的大小，选择较大的那个**。

具体来说：
- 如果 `当前元素 > 前面的子数组和 + 当前元素`，从当前元素重新开始
- 如果 `当前元素 ≤ 前面的子数组和 + 当前元素`，继续扩展子数组

为什么？因为选择较大的那个能让我们找到最大和。

### 2. 算法步骤

```
1. 初始化：max_current = nums[0]，max_global = nums[0]
2. 遍历数组（从第二个元素开始）：
   a. max_current = max(当前元素, max_current + 当前元素)
      - 要么从当前元素重新开始
      - 要么继续前面的子数组
   b. max_global = max(max_global, max_current)
      - 更新全局最大和
3. 返回 max_global
```

### 3. 动态规划理解

令 `dp[i]` 表示以第 `i` 个元素结尾的最大子数组和。

状态转移方程：
```
dp[i] = max(nums[i], dp[i-1] + nums[i])
```

解释：
- 如果 `dp[i-1] + nums[i] > nums[i]`，说明前面的正和有帮助，扩展子数组
- 否则，从当前元素重新开始

Kadane算法将dp数组优化为只用两个变量（空间复杂度从O(n)降到O(1)）。

### 4. 代码实现

#### Python实现

```python
from typing import List

class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        """
        最大子数组和 - Kadane算法（动态规划/贪心）

        时间复杂度：O(n)
        空间复杂度：O(1)

        算法思想：
            1. 如果前面的子数组和是负数，丢弃它（对当前和没有正向贡献）
            2. 从当前元素重新开始，或加上前面积累的和
            3. 维护全局最大和
        """
        max_current = nums[0]
        max_global = nums[0]

        for i in range(1, len(nums)):
            max_current = max(nums[i], max_current + nums[i])
            max_global = max(max_global, max_current)

        return max_global
```

#### C++实现

### 6. 核心代码详解

```python
for i in range(1, len(nums)):
    # 关键1：max_current 的更新
    # 如果前面的和是负数，就从当前元素重新开始
    # 如果前面的和是正数，就加上当前元素
    max_current = max(nums[i], max_current + nums[i])
    
    # 关键2：max_global 的更新
    # 记录遍历过程中的最大和
    max_global = max(max_global, max_current)
```

### 7. 为什么Kadane算法有效？

```
关键洞察：

  1. 贪心选择：
     - 比较 nums[i] 和 max_current + nums[i]
     - 选择较大的那个
     - 每一步都做最优选择
  
  2. 直观理解：
     - 如果 nums[i] > max_current + nums[i]，说明前面的子数组和是负贡献，丢弃
     - 如果 nums[i] ≤ max_current + nums[i]，说明前面的子数组和有帮助，保留
     - 不是简单判断前面的和是否为负，而是比较两种选择的大小
  
  3. 动态维护：
     - 每一步都维护当前最大和（max_current）
     - 每一步都维护全局最大和（max_global）
     - 最终得到全局最优解（最大和）
  
  4. 数学证明：
     - dp[i] = 以第 i 个元素结尾的最大子数组和
     - dp[i] = max(nums[i], dp[i-1] + nums[i])
     - Kadane算法将 dp[i-1] 优化为 max_current，将 dp 数组优化为两个变量
```

## 四. 解法二：分治法

### 1. 算法思想

分治法将数组分成两半，最大子数组有三种可能：
1. 完全在左半部分
2. 完全在右半部分
3. 跨越中点的子数组

### 2. 代码实现

```python
from typing import List

class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        """
        最大子数组和 - 分治法

        时间复杂度：O(n log n)
        空间复杂度：O(log n) - 递归栈空间
        """
        def helper(left: int, right: int) -> int:
            if left == right:
                return nums[left]
            
            mid = (left + right) // 2
            
            left_max = helper(left, mid)
            right_max = helper(mid + 1, right)
            
            left_sum = float('-inf')
            current_sum = 0
            for i in range(mid, left - 1, -1):
                current_sum += nums[i]
                left_sum = max(left_sum, current_sum)
            
            right_sum = float('-inf')
            current_sum = 0
            for i in range(mid + 1, right + 1):
                current_sum += nums[i]
                right_sum = max(right_sum, current_sum)
            
            cross_max = left_sum + right_sum
            
            return max(left_max, right_max, cross_max)
        
        return helper(0, len(nums) - 1)
```

```
数组 nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]

分治过程：

第一层：[0, 8]
  中点：mid = 4
  
  递归左半部分 [0, 4]
  递归右半部分 [5, 8]
  
第二层：
  左半部分 [0, 4]:
    中点：mid = 2
    
    递归 [0, 2]
    递归 [3, 4]
  
  右半部分 [5, 8]:
    中点：mid = 6
    
    递归 [5, 6]
    递归 [7, 8]

继续递归直到只剩一个元素...
```

### 4. 为什么分治法有效？

```
分治策略：

  1. 分解：将问题分解为规模更小的子问题
  2. 解决：递归求解子问题
  3. 合并：将子问题的解合并为原问题的解
  
  最大子数组的三种情况：
  
    情况1：在左半部分
      [-2, 1, -3] [4, -1] ...
              ↑        ↑
             left     mid
             
    情况2：在右半部分
      ... [4, -1] [2, 1, -5, 4]
             ↑         ↑      ↑
            mid+1     right
             
    情况3：跨越中点
      [-2, 1, -3, 4, -1] [2, 1, -5, 4]
                     ↑
                    mid
      
      需要找：
      - 以 mid 结尾的最大和（左半部分）
      - 以 mid+1 开头 的最大和（右半部分）
```

### Kadane算法（动态规划）

```
时间复杂度：O(n)
  - 只需遍历数组一次
  - 每次操作是常数时间

空间复杂度：O(1)
  - 只需要两个变量存储状态
  - 与输入规模无关
```

```
空间复杂度：O(log n)
  - 递归栈的深度为 log n
```

## 七. 深入理解Kadane算法

### 1. 状态转移方程

动态规划状态定义：
```
dp[i] = 以第 i 个元素结尾的最大子数组和
```

状态转移：
```
dp[i] = max(nums[i], dp[i-1] + nums[i])
```

初始条件：
```
dp[0] = nums[0]
```

### 2. 状态空间优化

原始动态规划需要O(n)空间存储dp数组：
```python
dp = [0] * len(nums)
dp[0] = nums[0]
for i in range(1, len(nums)):
    dp[i] = max(nums[i], dp[i-1] + nums[i])
```

Kadane算法观察到：
- 计算 dp[i] 只需要 dp[i-1]
- 所以只需要一个变量 max_current
- 最终答案是 max(dp[i])，可以用 max_global 维护

### 3. 直观理解

```
类比：寻找最大收益的连续交易日

  情况1：遇到负面消息就重新开始
    - 如果之前的累计收益是负数，说明之前的投资是亏损的
    - 应该止损，从当前重新开始投资
    - nums[i] = max(重新开始, 继续持有)
  
  情况2：持续持有
    - 如果之前的累计收益是正数，说明投资有回报
    - 可以继续持有，观察未来走势
    - 更新最大收益记录
```

### 4. 关键点总结

```
Kadane算法的三个关键点：

  1. 贪心选择：
     - 如果 max_current 是负数，丢弃它
     - 从当前元素重新开始
  
  2. 最优子结构：
     - dp[i] 的最优解只依赖于 dp[i-1] 的最优解
     - 满足最优子结构性质
  
  3. 全局最优：
     - 每一步都维护全局最大和
     - 最终得到整个问题的最优解
```

## 十. 总结

### 算法对比

| 解法 | 时间复杂度 | 空间复杂度 | 说明 |
|------|-----------|-----------|------|
| Kadane算法 | O(n) | O(1) | **推荐**，最优解 |
| 分治法 | O(n log n) | O(log n) | 符合进阶要求 |
| 暴力枚举 | O(n²) | O(1) | 简单但效率低 |

### 关键点总结

1. **核心洞察**：比较"当前元素"和"前面的子数组和+当前元素"，选择较大的
2. **动态规划**：dp[i] = max(nums[i], dp[i-1] + nums[i])
3. **空间优化**：Kadane算法将空间复杂度从O(n)降到O(1)
4. **时间复杂度**：Kadane算法是最优的O(n)解法
5. **正确理解**：不是"前面的和是负数就丢弃"，而是选择两种方案中较大的那个

```
   - 在更新max_global时记录窗口边界

4. 为什么叫"Kadane算法"？
   - 由Jay Kadane于1977年提出
   - 是解决最大子数组和问题的最优算法

5. 分治法的时间复杂度为什么是O(n log n)？
   - 递归深度是log n
   - 每次合并需要O(n)遍历跨越中点的元素
```
