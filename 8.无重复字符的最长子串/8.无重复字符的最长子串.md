# 3. 无重复字符的最长子串 - 详细解题分析

## 一. 题目

给定一个字符串 `s`，请你找出其中不含有重复字符的 **最长** 子串的长度。

### 示例

**示例 1：**
```
输入: s = "abcabcbb"
输出: 3
解释: 因为无重复字符的最长子串是 "abc" ，所以其长度为 3。
```

**示例 2：**
```
输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b" ，所以其长度为 1。
```

**示例 3：**
```
输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke" ，所以其长度为 3。
```

### 提示

- `0 <= s.length <= 5 * 10^4`
- `s` 由英文字母、数字、符号和空格组成

---

## 二. 解题思路

### 1. 问题分析

**目标**：找到不含有重复字符的最长子串的长度

**关键约束**：
- 子串必须是连续的
- 不能有重复字符
- 需要返回最大长度

**核心思想**：
- **暴力法**：枚举所有子串，检查是否有重复字符
- **滑动窗口法**：使用双指针和哈希表优化

---

### 解法一：暴力法（容易理解但低效）

#### 思路

枚举所有可能的子串，检查每个子串是否有重复字符。

```python
def lengthOfLongestSubstring(s: str) -> int:
    """
    无重复字符的最长子串 - 暴力法
    
    时间复杂度：O(n³)
    空间复杂度：O(min(n, m))，其中 m 是字符集大小
    
    说明：
        - 枚举所有子串
        - 对每个子串检查是否有重复字符
        - 代码简单但效率低
    """
    n = len(s)
    if n == 0:
        return 0
    
    max_length = 0
    
    # 枚举所有可能的子串
    for i in range(n):
        for j in range(i + 1, n + 1):
            # 检查子串 s[i:j] 是否有重复字符
            substring = s[i:j]
            if len(set(substring)) == len(substring):
                # 没有重复字符
                max_length = max(max_length, j - i)
    
    return max_length
```

#### 优缺点

- 优点：代码简单，容易理解
- 缺点：**时间复杂度 O(n³)**，对于大字符串效率低

### 解法二：滑动窗口法（推荐）

#### 思路

使用双指针维护一个滑动窗口，使用哈希表记录每个字符最后出现的位置。

```python
def lengthOfLongestSubstring(s: str) -> int:
    """
    无重复字符的最长子串 - 滑动窗口法（推荐）
    
    时间复杂度：O(n)
    空间复杂度：O(min(m, n))，其中 m 是字符集大小
    
    说明：
        - 使用字典记录每个字符最后出现的位置
        - 右指针遍历字符串
        - 当遇到重复字符时，移动左指针跳过重复字符
        - 维护最大长度
    """
    if not s:
        return 0
    
    # 使用字典记录每个字符最后出现的位置
    char_index = {}
    
    # 最大长度
    max_length = 0
    
    # 左指针
    left = 0
    
    # 遍历字符串（右指针）
    for right, char in enumerate(s):
        # 如果字符在当前窗口中出现过
        if char in char_index and char_index[char] >= left:
            # 移动左指针到重复字符的下一个位置
            left = char_index[char] + 1
        
        # 更新字符的位置
        char_index[char] = right
        
        # 更新最大长度
        max_length = max(max_length, right - left + 1)
    
    return max_length
```

#### 为什么滑动窗口法可行？

```
核心思想：
  
  1. 滑动窗口：维护一个没有重复字符的窗口
     - 左指针 left 指向窗口的起始位置
     - 右指针 right 遍历字符串

  2. 遇到重复字符时：
     - 如果重复字符在当前窗口内（char_index[char] >= left）
     - 将 left 移动到重复字符的下一个位置
     - 这样可以确保窗口内没有重复字符

  3. 更新最大长度：
     - 每次遍历后更新窗口长度
     - 记录最大长度

关键点：
  - 使用 char_index[char] >= left 判断字符是否在当前窗口内
  - 只有在窗口内的重复字符才需要移动 left
  - 不需要重新遍历，只移动 left 指针即可
```

### 解法三：优化版滑动窗口（更简洁）

#### 思路

使用 `collections.defaultdict` 记录字符位置，代码更简洁。

```python
from collections import defaultdict

def lengthOfLongestSubstring(s: str) -> int:
    """
    无重复字符的最长子串 - 优化版滑动窗口
    
    时间复杂度：O(n)
    空间复杂度：O(min(m, n))
    """
    if not s:
        return 0
    
    # 使用字典记录每个字符最后出现的位置
    char_index = defaultdict(int)
    
    max_length = 0
    left = 0
    
    for right, char in enumerate(s):
        # 如果字符在当前窗口中出现过
        if char in char_index and char_index[char] >= left:
            left = char_index[char] + 1
        
        # 更新字符的位置
        char_index[char] = right
        
        # 更新最大长度
        max_length = max(max_length, right - left + 1)
    
    return max_length
```

## 五. 小贴士

### 1. 为什么要判断 `char_index[char] >= left`？

```
问题：为什么要判断字符位置 >= left？

原因：确保重复字符在当前窗口内

例子：
  s = "abca"
  
  步骤1：处理完 "abc"，left = 0
  char_index = {'a': 0, 'b': 1, 'c': 2}
  
  步骤2：遇到 'a'（right = 3）
  char_index['a'] = 0
  
  判断1：'a' in char_index？是的
  判断2：char_index['a'] >= left？0 >= 0？是的！
  → 所以 'a' 是重复字符，需要移动 left
  
  如果不判断 left：
  → 会错误地认为 'a' 是重复字符
  → 但实际上 'a' 已经在窗口外了
```

### 2. 为什么是 `left = char_index[char] + 1`？

```
原因：需要将 left 移动到重复字符的下一个位置

例子：
  s = "abca"
  
  遇到 'a'（right = 3）
  char_index['a'] = 0
  
  移动策略：
  → left = char_index['a'] + 1 = 1
  → 新窗口：[1, 3] = "bca"
  → 没有重复字符
  
  如果 left = char_index['a'] = 0：
  → 新窗口：[0, 3] = "abca"
  → 仍然有重复字符 'a'
  → 错误！
```

### 3. 为什么是 `right - left + 1`？

```
原因：计算窗口长度

例子：
  left = 1, right = 3
  窗口：[1, 3] = "bca"
  长度 = 3 - 1 + 1 = 3 ✓
  
  如果不加 1：
  → 长度 = 3 - 1 = 2 ✗
  → 错误！
```

class Solution:

### Python（暴力法 - 容易理解）

```python
from typing import List

class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        """
        无重复字符的最长子串 - 暴力法
        
        时间复杂度：O(n³)
        空间复杂度：O(min(n, m))
        """
        n = len(s)
        if n == 0:
            return 0
        
        max_length = 0
        
        for i in range(n):
            for j in range(i + 1, n + 1):
                substring = s[i:j]
                if len(set(substring)) == len(substring):
                    max_length = max(max_length, j - i)
        
        return max_length
```

### C++（滑动窗口法）

```cpp
#include <string>
#include <unordered_map>

class Solution {
public:
    /**
     * 无重复字符的最长子串 - 滑动窗口法
     * 
     * 时间复杂度：O(n)
     * 空间复杂度：O(min(m, n))
     */
    int lengthOfLongestSubstring(string s) {
        if (s.empty()) {
            return 0;
        }
        
        std::unordered_map<char, int> char_index;
        int max_length = 0;
        int left = 0;
        
        for (int right = 0; right < s.length(); right++) {
            char c = s[right];
            
            if (char_index.find(c) != char_index.end() && char_index[c] >= left) {
                left = char_index[c] + 1;
            }
            
            char_index[c] = right;
            max_length = std::max(max_length, right - left + 1);
        }
        
        return max_length;
    }
};
```

## 八. 总结

### 核心要点

1. **滑动窗口**：使用双指针维护一个窗口
2. **哈希表**：记录每个字符最后出现的位置
3. **指针移动**：遇到重复字符时，移动左指针跳过重复字符
4. **边界判断**：`char_index[char] >= left` 确保重复字符在窗口内

### 算法模板
