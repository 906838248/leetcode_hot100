# 9. 找到字符串中所有字母异位词 - 详细解题分析

## 一. 题目

给定两个字符串 `s` 和 `p`，找到 `s` 中所有 `p` 的异位词的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

### 示例

**示例 1：**
```
输入: s = "cbaebabacd", p = "abc"
输出: [0,6]
解释: 
  起始索引等于 0 的子串是 "cba", 它是 "abc" 的异位词。
  起始索引等于 6 的子串是 "bac", 它是 "abc" 的异位词。
```

**示例 2：**
```
输入: s = "abab", p = "ab"
输出: [0,1,2]
解释: 
  起始索引等于 0 的子串是 "ab", 它是 "ab" 的异位词。
  起始索引等于 1 的子串是 "ba", 它是 "ab" 的异位词。
  起始索引等于 2 的子串是 "ab", 它是 "ab" 的异位词。
```

### 提示

- `1 <= s.length, p.length <= 3 * 10^4`
- `s` 和 `p` 仅包含小写字母

---

## 二. 解题思路

### 1. 问题分析

**目标**：找到 `s` 中所有 `p` 的异位词（字母异位词）的起始位置

**关键约束**：
- 异位词：两个字符串包含相同字符，但字符顺序不同
- 例如："abc" 和 "bca" 是异位词
- 需要找到所有长度为 `len(p)` 的异位词子串

**核心思想**：
- **滑动窗口法**：维护一个固定大小的窗口
- **哈希表**：记录字符频率，比较两个字符串是否相等

---

### 解法一：滑动窗口法（推荐）

#### 思路

使用固定大小的滑动窗口，使用字典记录字符频率，比较窗口和目标字符串是否相等。

```python
from typing import List
from collections import defaultdict

def findAnagrams(s: str, p: str) -> List[int]:
    """
    找到字符串中所有字母异位词 - 滑动窗口法
    
    时间复杂度：O(n)
    空间复杂度：O(1)，因为只有26个小写字母
    
    说明：
        - 使用字典记录目标字符串 p 的字符频率
        - 维护一个同样大小的滑动窗口
        - 比较两个字典是否相等
    """
    if len(p) > len(s):
        return []
    
    result = []
    dict_p = defaultdict(int)
    window = defaultdict(int)
    
    # 初始化目标字符串的字符频率
    for char in p:
        dict_p[char] += 1
    
    # 滑动窗口
    left = 0
    for right, char in enumerate(s):
        # 将当前字符加入窗口
        window[char] += 1
        
        # 如果窗口大小超过 p 的长度，移除最左边的字符
        if right - left + 1 > len(p):
            window[s[left]] -= 1
            if window[s[left]] == 0:
                window.pop(s[left])
            left += 1
        
        # 比较窗口和目标字符串是否相等
        if window == dict_p:
            result.append(left)
    
    return result
```

#### 为什么滑动窗口法可行？

```
核心思想：
  
  1. 异位词的特性：
     - 两个字符串包含相同的字符
     - 每个字符出现的频率相同
     - 字符顺序可以不同

  2. 如何判断异位词？
     - 使用字典记录字符频率
     - 比较两个字典是否相等

  3. 滑动窗口的优势：
     - 固定窗口大小：len(p)
     - 每次只移动一个字符
     - 无需重新计算整个窗口的频率
     - 只更新一个字符的频率即可
```

### 解法二：优化版滑动窗口（使用数组）

#### 思路

由于只包含小写字母，可以使用固定大小的数组代替字典，提高效率。

```python
from typing import List

def findAnagrams(s: str, p: str) -> List[int]:
    """
    找到字符串中所有字母异位词 - 优化版滑动窗口
    
    时间复杂度：O(n)
    空间复杂度：O(1)，只有26个字母
    """
    if len(p) > len(s):
        return []
    
    result = []
    
    # 使用数组记录字符频率
    target = [0] * 26
    window = [0] * 26
    
    # 初始化目标字符串的字符频率
    for char in p:
        target[ord(char) - ord('a')] += 1
    
    # 滑动窗口
    left = 0
    for right, char in enumerate(s):
        # 将当前字符加入窗口
        window[ord(char) - ord('a')] += 1
        
        # 如果窗口大小超过 p 的长度，移除最左边的字符
        if right - left + 1 > len(p):
            window[ord(s[left]) - ord('a')] -= 1
            left += 1
        
        # 比较窗口和目标字符串是否相等
        if window == target:
            result.append(left)
    
    return result
```

### 解法三：暴力法（容易理解但低效）

#### 思路

枚举所有可能的子串，对每个子串排序后比较。

```python
from typing import List

def findAnagrams(s: str, p: str) -> List[int]:
    """
    找到字符串中所有字母异位词 - 暴力法
    
    时间复杂度：O(n * m * log(m))，其中 m = len(p)
    空间复杂度：O(m)
    """
    if len(p) > len(s):
        return []
    
    result = []
    sorted_p = ''.join(sorted(p))
    
    # 枚举所有长度为 len(p) 的子串
    for i in range(len(s) - len(p) + 1):
        substring = s[i:i + len(p)]
        if ''.join(sorted(substring)) == sorted_p:
            result.append(i)
    
    return result
```

#### 优缺点

- 优点：代码简单，容易理解
- 缺点：**时间复杂度高**，排序操作开销大

## 三. 方法对比

| 方法 | 时间复杂度 | 空间复杂度 | 实现难度 | 推荐程度 |
|------|-----------|-----------|---------|---------|
| **暴力法** | O(n * m * log(m)) | O(m) | 简单 | 不推荐 |
| **滑动窗口法（字典）** | O(n) | O(m) | 中等 | **推荐** |
| **滑动窗口法（数组）** | O(n) | O(1) | 中等 | **强烈推荐** |

### 复杂度分析

```
暴力法：
  - 时间：O(n * m * log(m)) - 枚举子串 + 排序
  - 空间：O(m) - 排序字符串
  - n = 30000, m = 26 时，约 30000 * 26 * log(26) ≈ 30万次运算

滑动窗口法：
  - 时间：O(n) - 单次遍历
  - 空间：O(m) 或 O(1)
  - n = 30000 时，约 3万次运算

性能：滑动窗口法比暴力法快 10 倍以上！
```

## 四. 边界情况处理

```python
from typing import List
from collections import defaultdict

def findAnagrams(s: str, p: str) -> List[int]:
    """找到字符串中所有字母异位词 - 包含边界情况处理"""
    
    # 情况1：p 的长度大于 s
    if len(p) > len(s):
        return []
    
    # 情况2：空字符串
    if not s or not p:
        return []
    
    # 情况3：s 和 p 长度相等
    if len(s) == len(p):
        return [0] if sorted(s) == sorted(p) else []
    
    # 滑动窗口法
    result = []
    dict_p = defaultdict(int)
    window = defaultdict(int)
    
    for char in p:
        dict_p[char] += 1
    
    left = 0
    for right, char in enumerate(s):
        window[char] += 1
        
        if right - left + 1 > len(p):
            window[s[left]] -= 1
            if window[s[left]] == 0:
                window.pop(s[left])
            left += 1
        
        if window == dict_p:
            result.append(left)
    
    return result
```

## 五. 小贴士

### 1. 异位词和字母异位词的区别

```
异位词（Anagram）：
  - 两个字符串包含相同字符
  - 字符顺序可以不同
  - 例如："abc" 和 "bca" 是异位词

字母异位词：
  - 通常指包含相同字母的词
  - 在本题中，"abc" 和 "bca" 都是 "abc" 的字母异位词
```

### 2. 为什么用字典/数组而不是排序比较？

```
方法1：排序比较
  - 时间复杂度：O(m * log(m))
  - 每次比较都需要排序
  - 效率低

方法2：字典/数组比较
  - 时间复杂度：O(m)
  - 只记录字符频率
  - 效率高

示例：
  s = "cba", p = "abc"
  
  排序比较：
    sorted("cba") = "abc"
    sorted("abc") = "abc"
    "abc" == "abc" ✓
  
  字典比较：
    dict_s = {'c': 1, 'b': 1, 'a': 1}
    dict_p = {'a': 1, 'b': 1, 'c': 1}
    dict_s == dict_p ✓
```

### 3. 为什么要在窗口大小超过 p 时移除字符？

```
问题：为什么要移除字符？

原因：保持窗口大小始终为 len(p)

例子：
  s = "cbae", p = "abc"
  
  right = 0, char = 'c':
    window = {'c': 1}
    大小 = 1
  
  right = 1, char = 'b':
    window = {'c': 1, 'b': 1}
    大小 = 2
  
  right = 2, char = 'a':
    window = {'c': 1, 'b': 1, 'a': 1}
    大小 = 3
  
  right = 3, char = 'e':
    window = {'c': 1, 'b': 1, 'a': 1, 'e': 1}
    大小 = 4 > 3
    → 移除 s[left] = 'c'
    → window = {'b': 1, 'a': 1, 'e': 1}
    大小 = 3 ✓
```

## 六. 完整代码实现

### Python（滑动窗口法 - 字典版本）

```python
from typing import List
from collections import defaultdict

class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        """
        找到字符串中所有字母异位词 - 滑动窗口法
        
        时间复杂度：O(n)
        空间复杂度：O(m)，其中 m = len(p)
        
        算法步骤：
            1. 初始化目标字符串 p 的字符频率字典
            2. 使用滑动窗口遍历字符串 s
            3. 维护一个同样大小的窗口
            4. 比较窗口和目标字典是否相等
            5. 如果相等，添加起始索引到结果
        
        Args:
            s: 输入字符串
            p: 目标字符串
            
        Returns:
            List[int]：所有异位词的起始索引
        """
        if len(p) > len(s):
            return []
        
        result = []
        dict_p = defaultdict(int)
        window = defaultdict(int)
        
        for char in p:
            dict_p[char] += 1
        
        left = 0
        for right, char in enumerate(s):
            window[char] += 1
            
            if right - left + 1 > len(p):
                window[s[left]] -= 1
                if window[s[left]] == 0:
                    window.pop(s[left])
                left += 1
            
            if window == dict_p:
                result.append(left)
        
        return result
```

### C++（滑动窗口法 - 数组版本）

```cpp
#include <string>
#include <vector>

class Solution {
public:
    /**
     * 找到字符串中所有字母异位词 - 滑动窗口法
     * 
     * 时间复杂度：O(n)
     * 空间复杂度：O(1)，只有26个字母
     */
    std::vector<int> findAnagrams(std::string s, std::string p) {
        if (p.length() > s.length()) {
            return {};
        }
        
        std::vector<int> result;
        std::vector<int> target(26, 0);
        std::vector<int> window(26, 0);
        
        for (char c : p) {
            target[c - 'a']++;
        }
        
        int left = 0;
        for (int right = 0; right < s.length(); right++) {
            window[s[right] - 'a']++;
            
            if (right - left + 1 > p.length()) {
                window[s[left] - 'a']--;
                left++;
            }
            
            if (window == target) {
                result.push_back(left);
            }
        }
        
        return result;
    }
};
```

## 八. 总结

### 核心要点

1. **滑动窗口**：维护一个固定大小的窗口（长度为 `len(p)`）
2. **字符频率**：使用字典或数组记录字符出现次数
3. **比较相等**：比较窗口和目标字符串的字符频率是否相同
4. **移动窗口**：每次向右移动一格，同时更新窗口内容

### 算法模板
