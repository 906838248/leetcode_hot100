# 15. 轮转数组 - 详细解题分析

## 一. 题目

给定一个整数数组 `nums`，将数组中的元素向右轮转 `k` 个位置，其中 `k` 是非负数。

### 示例

**示例 1：**
```
输入: nums = [1,2,3,4,5,6,7], k = 3
输出: [5,6,7,1,2,3,4]
解释:
向右轮转 1 步: [7,1,2,3,4,5,6]
向右轮转 2 步: [6,7,1,2,3,4,5]
向右轮转 3 步: [5,6,7,1,2,3,4]
```

**示例 2：**
```
输入: nums = [-1,-100,3,99], k = 2
输出: [3,99,-1,-100]
解释:
向右轮转 1 步: [99,-1,-100,3]
向右轮转 2 步: [3,99,-1,-100]
```

### 提示

- `1 <= nums.length <= 10^5`
- `-2^31 <= nums[i] <= 2^31 - 1`
- `0 <= k <= 10^5`

**进阶**：
- 尽可能想出更多的解决方案，至少有 **三种** 不同的方法可以解决这个问题
- 你可以使用空间复杂度为 **O(1)** 的 **原地** 算法解决这个问题吗

---

## 二. 解题思路

### 1. 问题分析

**目标**：将数组元素向右移动 k 个位置

**关键约束**：
- 数组长度 n，k 可以大于 n（实际上只需要 k % n 次轮转）
- 需要原地操作，空间复杂度 O(1)
- 需要至少 3 种不同的解法

**核心思想**：
- **暴力法**：逐个轮转，简单但效率低
- **额外数组法**：使用额外空间，效率高但不满足进阶要求
- **三次翻转法**：利用翻转操作，原地 O(1) 空间
- **环状替换法**：数学方法，原地 O(1) 空间

---

## 三. 解法一：暴力法（逐个轮转）

### 1. 算法思想

每次向右轮转一位，执行 k 次。

**核心操作**：保存最后一个元素，将其他元素向右移动一位，将保存的元素放到开头。

### 2. 算法步骤

```
1. 处理 k > n 的情况：k = k % n
2. 重复 k 次：
   a. 保存最后一个元素：last = nums[n-1]
   b. 从后向前移动元素：nums[i] = nums[i-1]（i 从 n-1 到 1）
   c. 将保存的元素放到开头：nums[0] = last
3. 返回 nums
```

### 3. 代码实现

#### Python实现

```python
from typing import List

class Solution:
    def rotate(self, nums: List[int], k: int) -> None:
        """
        轮转数组 - 暴力法

        时间复杂度：O(n*k)
        空间复杂度：O(1)

        说明：每次轮转一个元素，重复 k 次
        """
        n = len(nums)
        if n == 0:
            return
        
        k = k % n
        if k == 0:
            return
        
        for _ in range(k):
            last = nums[-1]
            for i in range(n - 1, 0, -1):
                nums[i] = nums[i - 1]
            nums[0] = last
```

#### C++实现

```cpp
#include <vector>
using namespace std;

class Solution {
public:
    void rotate(vector<int>& nums, int k) {
        int n = nums.size();
        if (n == 0) return;
        
        k = k % n;
        if (k == 0) return;
        
        for (int step = 0; step < k; step++) {
            int last = nums[n - 1];
            for (int i = n - 1; i > 0; i--) {
                nums[i] = nums[i - 1];
            }
            nums[0] = last;
        }
    }
};
```

### 4. 图解过程

```
数组 nums = [1, 2, 3, 4, 5, 6, 7], k = 3

第1次轮转：
  [1, 2, 3, 4, 5, 6, 7]
                   ↓
  保存 last = 7
  从后向前移动：[1, 2, 3, 4, 5, 6, 6] → [1, 2, 3, 4, 5, 5, 6] → ... → [1, 1, 2, 3, 4, 5, 6]
  将 7 放到开头：[7, 1, 2, 3, 4, 5, 6]

第2次轮转：
  [7, 1, 2, 3, 4, 5, 6] → [6, 7, 1, 2, 3, 4, 5]

第3次轮转：
  [6, 7, 1, 2, 3, 4, 5] → [5, 6, 7, 1, 2, 3, 4]

最终结果：[5, 6, 7, 1, 2, 3, 4]
```

### 5. 复杂度分析

```
时间复杂度：O(n*k)
  - 外层循环执行 k 次
  - 内层循环每次移动 n-1 个元素
  - 最坏情况：k = n-1，时间复杂度 O(n²)

空间复杂度：O(1)
  - 只使用了常数个额外变量
  - 原地修改数组
```

## 四. 解法二：使用额外数组

### 1. 算法思想

创建一个新数组，将原数组的后 k 个元素放到新数组的前面，前 n-k 个元素放到后面。

**核心操作**：利用数组索引映射，直接将元素放到正确位置。

### 2. 算法步骤

```
1. 创建新数组 result，长度为 n
2. 处理 k > n 的情况：k = k % n
3. 将 nums[0..n-k-1] 复制到 result[k..n-1]
4. 将 nums[n-k..n-1] 复制到 result[0..k-1]
5. 将 result 复制回 nums
```

### 3. 代码实现

#### Python实现

```python
from typing import List

class Solution:
    def rotate(self, nums: List[int], k: int) -> None:
        """
        轮转数组 - 使用额外数组

        时间复杂度：O(n)
        空间复杂度：O(n)

        说明：将后 k 个元素放到前面，前 n-k 个元素放到后面
        """
        n = len(nums)
        if n == 0:
            return
        
        k = k % n
        if k == 0:
            return
        
        result = [0] * n
        for i in range(n):
            result[(i + k) % n] = nums[i]
        
        nums[:] = result
```

#### C++实现

```cpp
#include <vector>
using namespace std;

class Solution {
public:
    void rotate(vector<int>& nums, int k) {
        int n = nums.size();
        if (n == 0) return;
        
        k = k % n;
        if (k == 0) return;
        
        vector<int> result(n);
        for (int i = 0; i < n; i++) {
            result[(i + k) % n] = nums[i];
        }
        
        nums = result;
    }
};
```

### 4. 图解过程

```
数组 nums = [1, 2, 3, 4, 5, 6, 7], k = 3

步骤1：计算位置映射
  新数组索引 = (原数组索引 + k) % n

  i=0: (0+3)%7 = 3  → result[3] = nums[0] = 1
  i=1: (1+3)%7 = 4  → result[4] = nums[1] = 2
  i=2: (2+3)%7 = 5  → result[5] = nums[2] = 3
  i=3: (3+3)%7 = 6  → result[6] = nums[3] = 4
  i=4: (4+3)%7 = 0  → result[0] = nums[4] = 5
  i=5: (5+3)%7 = 1  → result[1] = nums[5] = 6
  i=6: (6+3)%7 = 2  → result[2] = nums[6] = 7

步骤2：result = [5, 6, 7, 1, 2, 3, 4]

最终结果：[5, 6, 7, 1, 2, 3, 4]
```

### 5. 复杂度分析

```
时间复杂度：O(n)
  - 只需要遍历数组一次
  - 每个元素只处理一次

空间复杂度：O(n)
  - 需要额外数组存储结果
  - 不满足进阶要求的 O(1) 空间
```

## 五. 解法三：三次翻转法【推荐 - 满足 O(1) 空间】

### 1. 算法思想

利用数组翻转操作：
- 整体翻转：使后 k 个元素移动到前面（但顺序相反）
- 再翻转前 k 个元素：恢复正确顺序
- 再翻转后 n-k 个元素：恢复正确顺序

**关键洞察**：翻转操作可以改变元素顺序，只需要 O(1) 额外空间。

### 2. 算法步骤

```
1. 处理 k > n 的情况：k = k % n
2. 如果 k == 0，直接返回
3. 整体翻转：[1,2,3,4,5,6,7] → [7,6,5,4,3,2,1]
4. 翻转前 k 个元素：[7,6,5] → [5,6,7]
5. 翻转后 n-k 个元素：[4,3,2,1] → [1,2,3,4]
6. 得到结果：[5,6,7,1,2,3,4]
```

### 3. 代码实现

#### Python实现

```python
from typing import List

class Solution:
    def rotate(self, nums: List[int], k: int) -> None:
        """
        轮转数组 - 三次翻转法

        时间复杂度：O(n)
        空间复杂度：O(1)

        算法思想：
          1. 整体翻转
          2. 翻转前 k 个元素
          3. 翻转后 n-k 个元素
        """
        n = len(nums)
        if n == 0:
            return
        
        k = k % n
        if k == 0:
            return
        
        def reverse(left: int, right: int) -> None:
            """翻转 nums[left..right]"""
            while left < right:
                nums[left], nums[right] = nums[right], nums[left]
                left += 1
                right -= 1
        
        reverse(0, n - 1)
        reverse(0, k - 1)
        reverse(k, n - 1)
```

#### C++实现

```cpp
#include <vector>
using namespace std;

class Solution {
public:
    void rotate(vector<int>& nums, int k) {
        int n = nums.size();
        if (n == 0) return;
        
        k = k % n;
        if (k == 0) return;
        
        auto reverse = [&](int left, int right) {
            while (left < right) {
                swap(nums[left], nums[right]);
                left++;
                right--;
            }
        };
        
        reverse(0, n - 1);
        reverse(0, k - 1);
        reverse(k, n - 1);
    }
};
```

### 4. 图解过程

```
数组 nums = [1, 2, 3, 4, 5, 6, 7], k = 3

步骤1：整体翻转 [0, 6]
  [1, 2, 3, 4, 5, 6, 7]
        ↓ 翻转
  [7, 6, 5, 4, 3, 2, 1]
  ↑_________________↑
       left=0, right=6

步骤2：翻转前 k 个元素 [0, 2]
  [7, 6, 5, 4, 3, 2, 1]
        ↓ 翻转
  [5, 6, 7, 4, 3, 2, 1]
  ↑_____↑
  left=0, right=2

步骤3：翻转后 n-k 个元素 [3, 6]
  [5, 6, 7, 4, 3, 2, 1]
              ↓ 翻转
  [5, 6, 7, 1, 2, 3, 4]
           ↑_________↑
           left=3, right=6

最终结果：[5, 6, 7, 1, 2, 3, 4]
```

### 5. 为什么三次翻转能正确工作？

```
直观理解：

  原数组：[1, 2, 3 | 4, 5, 6, 7]  （分为前后两部分）
           前半部分   后半部分

  目标：[4, 5, 6, 7 | 1, 2, 3]

  三次翻转的过程：
  
  1. 整体翻转：
     [1, 2, 3, 4, 5, 6, 7] → [7, 6, 5, 4, 3, 2, 1]
     
  2. 翻转前 k 个（目标是恢复后半部分）：
     [7, 6, 5 | 4, 3, 2, 1] → [5, 6, 7 | 4, 3, 2, 1]
     前 k 个变成了 [5,6,7]，但还需要是 [4,5,6,7]
     
  3. 翻转后 n-k 个（目标是恢复前半部分）：
     [5, 6, 7, 4 | 3, 2, 1] → [5, 6, 7, 1 | 2, 3, 4]
     后 n-k 个变成了 [1,2,3,4]

  最终：[5, 6, 7, 1, 2, 3, 4] ✓
```

### 6. 复杂度分析

```
时间复杂度：O(n)
  - 翻转操作总共遍历数组 3 次
  - 每次遍历 n 个元素，总共 3n 次操作
  - 常数因子不影响 O(n) 的复杂度

空间复杂度：O(1)
  - 只使用了常数个额外变量（left, right）
  - 原地修改数组
  - 满足进阶要求
```

## 六. 解法四：环状替换法【推荐 - O(1) 空间】

### 1. 算法思想

使用数学方法，计算每个元素应该去的新位置：
- 新位置 = (当前位置 + k) % n
- 从位置 0 开始，将元素移动到新位置
- 如果回到起点，说明当前位置的元素已经处理完，开始处理下一个未处理的位置
- 使用 gcd(n, k) 计算需要多少个独立的环

**核心操作**：将元素移动到它应该在的位置，形成多个环。

### 2. 算法步骤

```
1. 处理 k > n 的情况：k = k % n
2. 如果 k == 0，直接返回
3. 计算需要处理的环数：count = gcd(n, k)
4. 对每个环的起点（0 到 count-1）：
   a. current = start
   b. prev = nums[current]
   c. 循环直到回到 start：
      - next = (current + k) % n
      - temp = nums[next]
      - nums[next] = prev
      - prev = temp
      - current = next
```

### 3. 代码实现

#### Python实现

```python
from typing import List
import math

class Solution:
    def rotate(self, nums: List[int], k: int) -> None:
        """
        轮转数组 - 环状替换法

        时间复杂度：O(n)
        空间复杂度：O(1)

        算法思想：
          1. 计算 gcd(n, k) 确定需要处理的环数
          2. 对每个环进行元素替换
          3. 每个元素只被访问一次
        """
        n = len(nums)
        if n == 0:
            return
        
        k = k % n
        if k == 0:
            return
        
        count = math.gcd(n, k)
        
        for start in range(count):
            current = start
            prev = nums[current]
            
            while True:
                next_pos = (current + k) % n
                temp = nums[next_pos]
                nums[next_pos] = prev
                prev = temp
                current = next_pos
                
                if current == start:
                    break
```

#### C++实现

```cpp
#include <vector>
#include <numeric>
using namespace std;

class Solution {
public:
    void rotate(vector<int>& nums, int k) {
        int n = nums.size();
        if (n == 0) return;
        
        k = k % n;
        if (k == 0) return;
        
        int count = gcd(n, k);
        
        for (int start = 0; start < count; start++) {
            int current = start;
            int prev = nums[current];
            
            while (true) {
                int next_pos = (current + k) % n;
                int temp = nums[next_pos];
                nums[next_pos] = prev;
                prev = temp;
                current = next_pos;
                
                if (current == start) {
                    break;
                }
            }
        }
    }
};
```

### 4. 图解过程

```
数组 nums = [1, 2, 3, 4, 5, 6, 7], k = 3, n = 7

计算：gcd(7, 3) = 1
说明只需要处理 1 个环

从 start = 0 开始：

位置映射：
  0 → 3 → 6 → 2 → 5 → 1 → 4 → 0（回到起点）

步骤：
  位置 0: current=0, prev=1
          next_pos=(0+3)%7=3, temp=4
          nums[3]=1, prev=4, current=3
  
  位置 3: next_pos=(3+3)%7=6, temp=7
          nums[6]=4, prev=7, current=6
  
  位置 6: next_pos=(6+3)%7=2, temp=3
          nums[2]=7, prev=3, current=2
  
  位置 2: next_pos=(2+3)%7=5, temp=6
          nums[5]=3, prev=6, current=5
  
  位置 5: next_pos=(5+3)%7=1, temp=2
          nums[1]=6, prev=2, current=1
  
  位置 1: next_pos=(1+3)%7=4, temp=5
          nums[4]=2, prev=5, current=4
  
  位置 4: next_pos=(4+3)%7=0, temp=1
          nums[0]=5, prev=1, current=0
  
  current == start, 结束

最终结果：[5, 6, 7, 1, 2, 3, 4]
```

### 5. 为什么需要 gcd？

```
gcd 的作用：确定需要处理的独立环的数量

情况1：n=7, k=3, gcd(7,3)=1
  所有元素形成一个环
  路径：0 → 3 → 6 → 2 → 5 → 1 → 4 → 0

情况2：n=6, k=2, gcd(6,2)=2
  元素形成 2 个独立的环
  环1：0 → 2 → 4 → 0
  环2：1 → 3 → 5 → 1

情况3：n=6, k=3, gcd(6,3)=3
  元素形成 3 个独立的环
  环1：0 → 3 → 0
  环2：1 → 4 → 1
  环3：2 → 5 → 2

为什么需要多个环？
  - 如果只处理一个环，会陷入死循环
  - gcd(n, k) 给出了不重复的起点数量
  - 从每个起点开始，都能遍历该环的所有元素
```

### 6. 复杂度分析

```
时间复杂度：O(n)
  - 总共需要移动 n 个元素
  - 每个元素只被访问一次（移动到新位置）
  - 循环次数等于元素个数

空间复杂度：O(1)
  - 只使用了常数个额外变量（count, start, current, prev, next_pos, temp）
  - 原地修改数组
  - 满足进阶要求
```

```
  1. 三次翻转法：
     - 代码简单直观
     - 容易理解和实现
     - 面试时容易解释
  
  2. 环状替换法：
     - 数学方法，技巧性强
     - 每个元素只访问一次
     - 在某些场景下更高效
  
  3. 暴力法：
     - 简单但效率低
     - 适用于 k 很小的场景
  
  4. 额外数组法：
     - 最简单，但空间复杂度不满足要求
     - 适用于不关心空间的场景
```

## 九. 深入理解三次翻转法

### 1. 翻转操作的几何意义

```
翻转操作可以理解为：将数组分成两部分，然后交换位置

原始数组：[1, 2, 3 | 4, 5, 6, 7]
           前半部分   后半部分

目标：[4, 5, 6, 7 | 1, 2, 3]

三次翻转的过程可以这样理解：
  1. 整体翻转：将整个数组反转
     [1, 2, 3, 4, 5, 6, 7] → [7, 6, 5, 4, 3, 2, 1]
  
  2. 翻转前 k 个：使后半部分（现在是前半部分）恢复顺序
     [7, 6, 5 | 4, 3, 2, 1] → [5, 6, 7 | 4, 3, 2, 1]
  
  3. 翻转后 n-k 个：使前半部分（现在是后半部分）恢复顺序
     [5, 6, 7, 4 | 3, 2, 1] → [5, 6, 7, 1 | 2, 3, 4]
  
  结果：[5, 6, 7, 1, 2, 3, 4]
```

### 2. 代码实现细节

```python
def reverse(left: int, right: int) -> None:
    """翻转 nums[left..right]"""
    while left < right:
        nums[left], nums[right] = nums[right], nums[left]
        left += 1
        right -= 1

# 调用
reverse(0, n - 1)      # 整体翻转
reverse(0, k - 1)      # 翻转前 k 个
reverse(k, n - 1)      # 翻转后 n-k 个
```

### 3. 关键点总结

```
三次翻转法的三个关键点：

  1. 整体翻转：
     - 将整个数组反转
     - 后 k 个元素变成前 k 个（顺序相反）
  
  2. 翻转前 k 个：
     - 恢复前 k 个元素的正确顺序
     - 这些元素实际上是原来的后 k 个
  
  3. 翻转后 n-k 个：
     - 恢复后 n-k 个元素的正确顺序
     - 这些元素实际上是原来的前 n-k 个
```

## 十一. 完整代码汇总

### Python实现（四种方法）

```python
from typing import List
import math

class Solution:
    # 方法1：暴力法
    def rotate_v1(self, nums: List[int], k: int) -> None:
        n = len(nums)
        if n == 0:
            return
        k = k % n
        if k == 0:
            return
        for _ in range(k):
            last = nums[-1]
            for i in range(n - 1, 0, -1):
                nums[i] = nums[i - 1]
            nums[0] = last

    # 方法2：使用额外数组
    def rotate_v2(self, nums: List[int], k: int) -> None:
        n = len(nums)
        if n == 0:
            return
        k = k % n
        if k == 0:
            return
        result = [0] * n
        for i in range(n):
            result[(i + k) % n] = nums[i]
        nums[:] = result

    # 方法3：三次翻转法
    def rotate_v3(self, nums: List[int], k: int) -> None:
        n = len(nums)
        if n == 0:
            return
        k = k % n
        if k == 0:
            return
        def reverse(left: int, right: int) -> None:
            while left < right:
                nums[left], nums[right] = nums[right], nums[left]
                left += 1
                right -= 1
        reverse(0, n - 1)
        reverse(0, k - 1)
        reverse(k, n - 1)

    # 方法4：环状替换法
    def rotate_v4(self, nums: List[int], k: int) -> None:
        n = len(nums)
        if n == 0:
            return
        k = k % n
        if k == 0:
            return
        count = math.gcd(n, k)
        for start in range(count):
            current = start
            prev = nums[current]
            while True:
                next_pos = (current + k) % n
                temp = nums[next_pos]
                nums[next_pos] = prev
                prev = temp
                current = next_pos
                if current == start:
                    break
```

### C++实现（四种方法）

```cpp
#include <vector>
#include <numeric>
using namespace std;

class Solution {
public:
    // 方法1：暴力法
    void rotate_v1(vector<int>& nums, int k) {
        int n = nums.size();
        if (n == 0) return;
        k = k % n;
        if (k == 0) return;
        for (int step = 0; step < k; step++) {
            int last = nums[n - 1];
            for (int i = n - 1; i > 0; i--) {
                nums[i] = nums[i - 1];
            }
            nums[0] = last;
        }
    }

    // 方法2：使用额外数组
    void rotate_v2(vector<int>& nums, int k) {
        int n = nums.size();
        if (n == 0) return;
        k = k % n;
        if (k == 0) return;
        vector<int> result(n);
        for (int i = 0; i < n; i++) {
            result[(i + k) % n] = nums[i];
        }
        nums = result;
    }

    // 方法3：三次翻转法
    void rotate_v3(vector<int>& nums, int k) {
        int n = nums.size();
        if (n == 0) return;
        k = k % n;
        if (k == 0) return;
        auto reverse = [&](int left, int right) {
            while (left < right) {
                swap(nums[left], nums[right]);
                left++;
                right--;
            }
        };
        reverse(0, n - 1);
        reverse(0, k - 1);
        reverse(k, n - 1);
    }

    // 方法4：环状替换法
    void rotate_v4(vector<int>& nums, int k) {
        int n = nums.size();
        if (n == 0) return;
        k = k % n;
        if (k == 0) return;
        int count = gcd(n, k);
        for (int start = 0; start < count; start++) {
            int current = start;
            int prev = nums[current];
            while (true) {
                int next_pos = (current + k) % n;
                int temp = nums[next_pos];
                nums[next_pos] = prev;
                prev = temp;
                current = next_pos;
                if (current == start) {
                    break;
                }
            }
        }
    }
};
```

## 十二. 总结

### 算法对比

| 解法       | 时间复杂度  | 空间复杂度 | 说明           |
| -------- | ------ | ----- | ------------ |
| 暴力法      | O(n*k) | O(1)  | 简单但效率低       |
| 额外数组     | O(n)   | O(n)  | 空间复杂度不满足进阶要求 |
| **三次翻转** | O(n)   | O(1)  | **推荐**，代码直观  |
| **环状替换** | O(n)   | O(1)  | **推荐**，数学方法  |

### 关键点总结

1. **处理 k > n**：使用 `k = k % n` 避免不必要的轮转
2. **边界情况**：k = 0 或 k = n 时不需要轮转
3. **空间复杂度**：如果要求 O(1) 空间，选择三次翻转或环状替换
4. **代码简洁性**：三次翻转法最容易理解和实现
