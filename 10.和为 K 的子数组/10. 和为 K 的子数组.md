# 10. 和为 K 的子数组 - 详细解题分析

## 一. 题目

给你一个整数数组 `nums` 和一个整数 `k` ，请你统计并返回该数组中和为 `k` 的子数组的个数。

子数组是数组中元素的连续非空序列。

### 示例

**示例 1：**
```
输入: nums = [1,1,1], k = 2
输出: 2
解释: 
  子数组 [1,1]（索引 0-1）：1 + 1 = 2
  子数组 [1,1]（索引 1-2）：1 + 1 = 2
```

**示例 2：**
```
输入: nums = [1,2,3], k = 3
输出: 2
解释: 
  子数组 [1,2]（索引 0-1）：1 + 2 = 3
  子数组 [3]（索引 2-2）：3 = 3
```

### 提示

- `1 <= nums.length <= 2 * 10^4`
- `-1000 <= nums[i] <= 1000`
- `-10^7 <= k <= 10^7`

---

## 二. 解题思路

### 1. 问题分析

**目标**：统计和为 `k` 的连续子数组的个数

**关键约束**：
- 子数组必须是连续的
- 子数组是非空的
- 数组元素可以是负数（这使得问题变得更有挑战性）

**核心思想**：
- **前缀和**：计算到每个位置为止的累计和
- **哈希表**：记录前缀和出现的次数，快速查找满足条件的子数组

---

### 解法一：前缀和 + 哈希表法（推荐）

#### 思路

使用前缀和配合哈希表，在 O(n) 时间内解决问题。

```python
from typing import List
from collections import defaultdict

class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        """
        前缀和 + 哈希表法

        时间复杂度：O(n)
        空间复杂度：O(n)

        核心思想：
            - 计算前缀和：prefix[i] = nums[0] + nums[1] + ... + nums[i-1]
            - 子数组 [left, right] 的和 = prefix[right+1] - prefix[left]
            - 我们需要找 prefix[right+1] - prefix[left] == k
            - 即 prefix[left] == prefix[right+1] - k
            - 使用哈希表记录每个前缀和出现的次数
        """
        prefix_count = defaultdict(int)
        prefix_count[0] = 1

        prefix = 0
        result = 0

        for num in nums:
            prefix += num
            result += prefix_count[prefix - k]
            prefix_count[prefix] += 1

        return result
```

#### 图解过程

```
数组 nums = [1, 2, 3]，k = 3

目标：找到所有和为 3 的连续子数组

步骤1：初始化
  prefix_count = {0: 1}  # 空数组的前缀和为0，出现1次
  prefix = 0
  result = 0

步骤2：遍历 num = 1
  prefix = 0 + 1 = 1

  查找 prefix_count[prefix - k] = prefix_count[1 - 3] = prefix_count[-2] = 0
  result += 0

  prefix_count[1] += 1
  prefix_count = {0: 1, 1: 1}

  当前状态：
    nums: [1, 2, 3]
          ↑
          prefix = 1

步骤3：遍历 num = 2
  prefix = 1 + 2 = 3

  查找 prefix_count[3 - 3] = prefix_count[0] = 1
  result += 1  ← 找到一个子数组！子数组是 [1,2]

  prefix_count[3] += 1
  prefix_count = {0: 1, 1: 1, 3: 1}

  当前状态：
    nums: [1, 2, 3]
             ↑↑
          子数组 [1,2] 的和为 3

步骤4：遍历 num = 3
  prefix = 3 + 3 = 6

  查找 prefix_count[6 - 3] = prefix_count[3] = 1
  result += 1  ← 找到一个子数组！子数组是 [3]

  prefix_count[6] += 1
  prefix_count = {0: 1, 1: 1, 3: 1, 6: 1}

  当前状态：
    nums: [1, 2, 3]
                ↑
          子数组 [3] 的和为 3

最终结果：result = 2
```

#### 为什么前缀和 + 哈希表法可行？

```
核心思想：

  1. 前缀和的定义：
     prefix[i] = nums[0] + nums[1] + ... + nums[i-1]
     prefix[0] = 1（空数组的前缀和）

  2. 子数组和的计算：
     子数组 [left, right] 的和
     = nums[left] + nums[left+1] + ... + nums[right]
     = prefix[right+1] - prefix[left]

  3. 问题的转化：
     我们需要找：prefix[right+1] - prefix[left] == k
     整理得：prefix[left] == prefix[right+1] - k

  4. 哈希表的作用：
     - 记录每个前缀和出现的次数
     - 当遍历到位置 right 时，prefix[right+1] 是已知的
     - 我们需要找有多少个 left 满足：prefix[left] == prefix[right+1] - k
     - 即在哈希表中查找 prefix[right+1] - k 出现的次数

  5. 为什么从左到右遍历就能找到所有子数组？
     - 当遍历到位置 i 时，前缀和 prefix[i+1] 是确定的
     - 此时，所有以 i 结尾的子数组都已经考虑过了
     - 因为 prefix[i+1] - prefix[j] (j <= i) 就是子数组 [j, i] 的和
```

### 解法二：笨办法（暴力枚举）

#### 思路

枚举所有可能的子数组，计算它们的和，统计等于 k 的个数。

```python
from typing import List

class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        """
        暴力枚举法

        时间复杂度：O(n²)
        空间复杂度：O(1)

        说明：
            - 枚举所有可能的子数组
            - 计算每个子数组的和
            - 如果和等于 k，计数器加 1
        """
        result = 0
        n = len(nums)

        for i in range(n):
            total = 0
            for j in range(i, n):
                total += nums[j]
                if total == k:
                    result += 1

        return result
```

#### 复杂度分析

```
暴力枚举法：
  时间复杂度：O(n²)
    - 外层循环 n 次
    - 内层循环平均 n/2 次
    - 总共 n*(n+1)/2 ≈ O(n²)

  空间复杂度：O(1)
    - 只使用了常数个变量

前缀和 + 哈希表法：
  时间复杂度：O(n)
    - 只需遍历数组一次

  空间复杂度：O(n)
    - 哈希表最多存储 n+1 个不同的前缀和
```

### C++ 实现

```cpp
#include <unordered_map>
#include <vector>

using namespace std;

class Solution {
public:
    int subarraySum(vector<int>& nums, int k) {
        unordered_map<int, int> prefix_count;
        prefix_count[0] = 1;

        int prefix = 0;
        int result = 0;

        for (int num : nums) {
            prefix += num;
            result += prefix_count[prefix - k];
            prefix_count[prefix]++;
        }

        return result;
    }
};
```

## 五. 总结

| 解法 | 时间复杂度 | 空间复杂度 | 说明 |
|------|-----------|-----------|------|
| 前缀和 + 哈希表 | O(n) | O(n) | 推荐解法 |
| 暴力枚举 | O(n²) | O(1) | 简单但效率低 |

**关键点：**
1. 理解前缀和的概念和计算方式
2. 将子数组和问题转化为前缀和的差值问题
3. 使用哈希表快速查找满足条件的前缀和
