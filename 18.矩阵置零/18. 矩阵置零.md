# 18. 矩阵置零 - 详细解题分析

## 一. 题目

给定一个 `m x n` 的矩阵，如果一个元素为 0，则将其所在行和列的所有元素都设为 0。请使用原地算法。

### 示例

**示例 1：**
```
输入：matrix = [[1,1,1],[1,0,1],[1,1,1]]
输出：[[1,0,1],[0,0,0],[1,0,1]]
```

**示例 2：**
```
输入：matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]
输出：[[0,0,0,0],[0,4,5,0],[0,3,1,0]]
```

### 提示

- `m == matrix.length`
- `n == matrix[0].length`
- `1 <= m, n <= 200`
- `-2^31 <= matrix[i][j] <= 2^31 - 1`

**进阶**：
- 一个直观的解决方案是使用 O(mn) 的额外空间，但这并不是一个好的解决方案
- 一个简单的改进方案是使用 O(m + n) 的额外空间，但这仍然不是最好的解决方案
- 你能想出一个仅使用常量空间的解决方案吗？

---

## 二. 解题思路

### 1. 问题分析

**目标**：将矩阵中所有为 0 的元素所在的行和列全部设为 0

**关键约束**：
- 需要使用原地算法
- 进阶：O(1) 额外空间
- 矩阵大小：`1 <= m, n <= 200`

**核心思想**：
- **标记法**：用第一行和第一列来标记哪些行和列需要置零
- **两次遍历**：第一次标记，第二次置零
- 关键：需要先处理边界，再处理内部

### 2. 关键洞察

```
对于矩阵 matrix：
  [
    [1, 1, 1],
    [1, 0, 1],
    [1, 1, 1]
  ]

观察：
  - matrix[1][1] = 0
  - 需要将第 1 行和第 1 列全部置零

核心思想：
  - 使用第一行和第一列作为标记数组
  - 记录哪些行和列需要置零
  - 最后统一处理
```

## 三. 解法一：暴力法

### 1. 算法思想

找到每个 0，然后将其所在行和列全部置零。

**问题**：可能会重复处理同一个位置。

### 2. 代码实现

#### Python实现

```python
from typing import List

class Solution:
    def setZeroes(self, matrix: List[List[int]]) -> None:
        """
        矩阵置零 - 暴力法（不推荐）

        时间复杂度：O(m*n*(m+n))
        空间复杂度：O(1)

        说明：找到每个0，将其行和列置零
        """
        m = len(matrix)
        n = len(matrix[0])

        for i in range(m):
            for j in range(n):
                if matrix[i][j] == 0:
                    self.setRowAndColZero(matrix, i, j, m, n)

    def setRowAndColZero(self, matrix: List[List[int]], row: int, col: int, m: int, n: int):
        for c in range(n):
            matrix[row][c] = 0
        for r in range(m):
            matrix[r][col] = 0
```

### 3. 复杂度分析

```
时间复杂度：O(m*n*(m+n))
  - 外层循环：m*n 次
  - 每个 0 需要 O(m+n) 时间置零行和列
  - 最坏情况：所有元素都是 0

空间复杂度：O(1)
  - 只使用了常数个额外变量
  - 原地修改矩阵

结论：时间复杂度太高，不推荐
```

## 四. 解法二：标记数组法（O(m+n) 空间）

### 1. 算法思想

使用两个额外的数组来记录需要置零的行和列。

### 2. 算法步骤

```
1. 创建 rows[m] 和 cols[n]，初始化为 False
2. 第一次遍历：
   - 如果 matrix[i][j] == 0
   - 设置 rows[i] = True
   - 设置 cols[j] = True
3. 第二次遍历：
   - 如果 rows[i] 或 cols[j] 为 True
   - 设置 matrix[i][j] = 0
```

### 3. 代码实现

#### Python实现

```python
from typing import List

class Solution:
    def setZeroes(self, matrix: List[List[int]]) -> None:
        """
        矩阵置零 - 标记数组法

        时间复杂度：O(m*n)
        空间复杂度：O(m+n)

        说明：使用额外数组记录需要置零的行和列
        """
        m = len(matrix)
        n = len(matrix[0])

        rows = [False] * m
        cols = [False] * n

        for i in range(m):
            for j in range(n):
                if matrix[i][j] == 0:
                    rows[i] = True
                    cols[j] = True

        for i in range(m):
            for j in range(n):
                if rows[i] or cols[j]:
                    matrix[i][j] = 0
```

#### C++实现

```cpp
#include <vector>
using namespace std;

class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int m = matrix.size();
        int n = matrix[0].size();

        vector<bool> rows(m, false);
        vector<bool> cols(n, false);

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (matrix[i][j] == 0) {
                    rows[i] = true;
                    cols[j] = true;
                }
            }
        }

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (rows[i] || cols[j]) {
                    matrix[i][j] = 0;
                }
            }
        }
    }
};
```

### 4. 图解过程

```
矩阵 matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]], m=3, n=4

第一步：标记
  遍历矩阵，找到所有 0 的位置
  matrix[0][0] = 0 → rows[0] = True, cols[0] = True
  matrix[0][3] = 0 → rows[0] = True, cols[3] = True

  rows = [True, False, False]
  cols = [True, False, False, True]

第二步：置零
  遍历矩阵，如果 rows[i] 或 cols[j] 为 True，置零

  i=0: cols[0] 和 cols[3] 为 True
    [0, 0, 0, 0]
  
  i=1: cols[0] 和 cols[3] 为 True
    [0, 4, 5, 0]
  
  i=2: cols[0] 和 cols[3] 为 True
    [0, 3, 1, 0]

最终结果：[[0,0,0,0],[0,4,5,0],[0,3,1,0]]
```

### 5. 复杂度分析

```
时间复杂度：O(m*n)
  - 第一次遍历：m*n 次
  - 第二次遍历：m*n 次
  - 总共 2*m*n 次操作

空间复杂度：O(m+n)
  - rows 数组：m 个 bool
  - cols 数组：n 个 bool
  - 总共 m+n 个 bool

结论：满足 O(m+n) 空间，但不满足进阶要求
```

## 五. 解法三：原地算法（O(1) 空间）【推荐】

### 1. 算法思想

利用矩阵的第一行和第一列作为标记数组：
- 检查第一行和第一列是否有 0（需要特殊处理）
- 用第一行和第一列来标记其他行和列是否有 0
- 先处理内部元素，再处理第一行和第一列

### 2. 算法步骤

```
1. 检查第一行和第一列是否有 0
   - first_row_has_zero
   - first_col_has_zero
2. 用第一行和第一列标记其他行和列：
   - 对于 i=1 到 m-1，j=1 到 n-1：
     - 如果 matrix[i][j] == 0
     - 设置 matrix[i][0] = 0（标记行）
     - 设置 matrix[0][j] = 0（标记列）
3. 根据标记置零其他行和列：
   - 对于 i=1 到 m-1，j=1 到 n-1：
     - 如果 matrix[i][0] == 0 或 matrix[0][j] == 0
     - 设置 matrix[i][j] = 0
4. 处理第一行和第一列：
   - 如果 first_row_has_zero，置零第一行
   - 如果 first_col_has_zero，置零第一列
```

### 3. 代码实现

#### Python实现

```python
from typing import List

class Solution:
    def setZeroes(self, matrix: List[List[int]]) -> None:
        """
        矩阵置零 - 原地算法

        时间复杂度：O(m*n)
        空间复杂度：O(1)

        算法思想：
          1. 用第一行和第一列作为标记
          2. 先处理内部元素，再处理第一行和第一列
        """
        if not matrix or not matrix[0]:
            return

        m = len(matrix)
        n = len(matrix[0])

        first_row_has_zero = False
        first_col_has_zero = False

        for j in range(n):
            if matrix[0][j] == 0:
                first_row_has_zero = True
                break

        for i in range(m):
            if matrix[i][0] == 0:
                first_col_has_zero = True
                break

        for i in range(1, m):
            for j in range(1, n):
                if matrix[i][j] == 0:
                    matrix[i][0] = 0
                    matrix[0][j] = 0

        for i in range(1, m):
            for j in range(1, n):
                if matrix[i][0] == 0 or matrix[0][j] == 0:
                    matrix[i][j] = 0

        if first_row_has_zero:
            for j in range(n):
                matrix[0][j] = 0

        if first_col_has_zero:
            for i in range(m):
                matrix[i][0] = 0
```

#### C++实现

```cpp
#include <vector>
using namespace std;

class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        if (matrix.empty() || matrix[0].empty()) {
            return;
        }

        int m = matrix.size();
        int n = matrix[0].size();

        bool first_row_has_zero = false;
        bool first_col_has_zero = false;

        for (int j = 0; j < n; j++) {
            if (matrix[0][j] == 0) {
                first_row_has_zero = true;
                break;
            }
        }

        for (int i = 0; i < m; i++) {
            if (matrix[i][0] == 0) {
                first_col_has_zero = true;
                break;
            }
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][j] == 0) {
                    matrix[i][0] = 0;
                    matrix[0][j] = 0;
                }
            }
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][0] == 0 || matrix[0][j] == 0) {
                    matrix[i][j] = 0;
                }
            }
        }

        if (first_row_has_zero) {
            for (int j = 0; j < n; j++) {
                matrix[0][j] = 0;
            }
        }

        if (first_col_has_zero) {
            for (int i = 0; i < m; i++) {
                matrix[i][0] = 0;
            }
        }
    }
};
```

```

  - 利用第一行和第一列作为标记
  - 满足进阶要求
```

## 六. 原地算法的深入理解

### 1. 为什么需要特殊处理第一行和第一列？

```
第一行和第一列被用作标记，不能直接置零

问题：
  如果直接置零第一行或第一列，会丢失原始信息
  
例如：
  matrix = [[0, 1], [1, 1]]
  
  如果第一步就置零第一行：
    matrix = [[0, 0], [1, 1]]
    第一列的标记信息丢失了
  
正确做法：
  1. 先检查第一行和第一列是否有 0（记录到变量）
  2. 用第一行和第一列标记其他位置
  3. 处理完其他位置后，再处理第一行和第一列
```

### 2. 为什么从 (1,1) 开始遍历？

```
从 (1,1) 开始是因为：
  - (0,0) 被两个方向共用，不能直接用作单一方向的标记
  - 第一行和第一列需要特殊处理

遍历范围：
  - i 从 1 到 m-1
  - j 从 1 到 n-1
  - 不包括第一行和第一列
```

### 3. 标记的原理

```
对于 matrix[i][j] = 0（i>0, j>0）：
  - 设置 matrix[i][0] = 0（标记第 i 行需要置零）
  - 设置 matrix[0][j] = 0（标记第 j 列需要置零）

验证：
  matrix = [[1,1,1],
            [1,0,1],
            [1,1,1]]
  
  matrix[1][1] = 0
  设置 matrix[1][0] = 0（标记第 1 行）
  设置 matrix[0][1] = 0（标记第 1 列）
  
  matrix = [[1,0,1],
            [0,0,1],
            [1,1,1]]
```

### 4. 代码优化

```python
# 简化版本
def setZeroes(self, matrix: List[List[int]]) -> None:
    m, n = len(matrix), len(matrix[0])
    
    first_row_zero = any(matrix[0][j] == 0 for j in range(n))
    first_col_zero = any(matrix[i][0] == 0 for i in range(m))
    
    for i in range(1, m):
        for j in range(1, n):
            if matrix[i][j] == 0:
                matrix[i][0] = matrix[0][j] = 0
    
    for i in range(1, m):
        for j in range(1, n):
            if matrix[i][0] == 0 or matrix[0][j] == 0:
                matrix[i][j] = 0
    
    if first_row_zero:
        matrix[0] = [0] * n
    if first_col_zero:
        for i in range(m):
            matrix[i][0] = 0
```

```python
        空间复杂度：O(1)

        关键点：
          - 需要特殊处理第一行和第一列
          - 因为它们被用作标记，不能直接置零
          - 先检查并记录，最后统一处理

        为什么从 (1,1) 开始？
          - (0,0) 被两个方向共用
          - 需要先检查第一行和第一列是否有 0
        """
        if not matrix or not matrix[0]:
            return

        m = len(matrix)
        n = len(matrix[0])

        first_row_has_zero = False
        first_col_has_zero = False

        for j in range(n):
            if matrix[0][j] == 0:
                first_row_has_zero = True
                break

        for i in range(m):
            if matrix[i][0] == 0:
                first_col_has_zero = True
                break

        for i in range(1, m):
            for j in range(1, n):
                if matrix[i][j] == 0:
                    matrix[i][0] = 0
                    matrix[0][j] = 0

        for i in range(1, m):
            for j in range(1, n):
                if matrix[i][0] == 0 or matrix[0][j] == 0:
                    matrix[i][j] = 0

        if first_row_has_zero:
            for j in range(n):
                matrix[0][j] = 0

        if first_col_has_zero:
            for i in range(m):
                matrix[i][0] = 0
```

```cpp
#include <vector>
using namespace std;

class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        if (matrix.empty() || matrix[0].empty()) {
            return;
        }

        int m = matrix.size();
        int n = matrix[0].size();

        bool first_row_has_zero = false;
        bool first_col_has_zero = false;

        for (int j = 0; j < n; j++) {
            if (matrix[0][j] == 0) {
                first_row_has_zero = true;
                break;
            }
        }

        for (int i = 0; i < m; i++) {
            if (matrix[i][0] == 0) {
                first_col_has_zero = true;
                break;
            }
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][j] == 0) {
                    matrix[i][0] = 0;
                    matrix[0][j] = 0;
                }
            }
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][0] == 0 || matrix[0][j] == 0) {
                    matrix[i][j] = 0;
                }
            }
        }

        if (first_row_has_zero) {
            for (int j = 0; j < n; j++) {
                matrix[0][j] = 0;
            }
        }

        if (first_col_has_zero) {
            for (int i = 0; i < m; i++) {
                matrix[i][0] = 0;
            }
        }
    }
};
```

### 为什么原地算法最优？

```
1. 时间复杂度 O(m*n)：
   - 多次遍历矩阵
   - 每次遍历都是 O(m*n)
   - 总共 O(m*n)

2. 空间复杂度 O(1)：
   - 只使用常数个额外变量
   - 利用第一行和第一列作为标记
   - 满足进阶要求

3. 原地操作：
   - 不需要额外的数据结构
   - 不需要额外的存储空间
   - 直接在原矩阵上修改

4. 代码清晰：
   - 分步骤处理，逻辑清晰
   - 特殊处理边界情况
   - 易于理解和实现
```

## 十二. 面试要点

```
面试官可能会问的问题：

1. 为什么不能用暴力法？
   - 时间复杂度 O(m*n*(m+n)) 太高
   - 最坏情况需要处理 m*n 个 0
   - 每个 0 需要 O(m+n) 时间置零

2. 为什么需要 O(m+n) 的额外空间？
   - 需要记录哪些行和列需要置零
   - 不能在原矩阵上直接标记（会丢失信息）

3. 如何在 O(1) 空间内完成？
   - 利用第一行和第一列作为标记
   - 需要特殊处理第一行和第一列
   - 先检查并记录，再统一处理

4. 为什么从 (1,1) 开始遍历？
   - (0,0) 被两个方向共用
   - 第一行和第一列需要特殊处理
   - 需要先检查它们是否有 0

5. 如何处理边界情况？
   - 空矩阵：直接返回
   - 单行单列：特殊处理
   - 第一行/列有 0：需要记录最后处理
```

## 十三. 总结

### 核心算法

```
原地算法：
  1. 检查第一行和第一列是否有 0
  2. 用第一行和第一列标记其他位置
  3. 根据标记置零其他位置
  4. 处理第一行和第一列

关键点：
  - O(m*n) 时间复杂度
  - O(1) 空间复杂度
  - 利用第一行和第一列作为标记
  - 需要特殊处理第一行和第一列
```

### 关键点总结

1. **标记法**：用第一行和第一列作为标记
2. **分步处理**：先内部后边界
3. **特殊处理**：第一行和第一列需要记录后处理
4. **两次遍历**：标记和置零分开进行

### 最终建议

```
对于矩阵置零这个问题：

  ✅ 使用原地算法（O(m*n)时间，O(1)空间）
  ✅ 利用第一行和第一列作为标记
  ✅ 分步骤处理，逻辑清晰
  ✅ 代码简洁高效
  
  需要理解的要点：
  ✅ 为什么需要特殊处理第一行和第一列
  ✅ 为什么从 (1,1) 开始遍历
  ✅ 如何记录和处理边界情况
  ✅ 标记法的原理
  
  需要避免的方法：
  ❌ 暴力法：时间复杂度太高
  ❌ O(m+n) 空间：不满足进阶要求
```
